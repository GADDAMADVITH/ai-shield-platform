# AI Shield

Enterprise AI security platform — continuous vulnerability scanning, live threat telemetry, and executive-ready reports.

## Development

Requires Node.js. Prefer [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) if you need to manage versions.

```sh
git clone <this-repository-url>
cd <repository-name>
npm i
npm run dev
```

## Built with

- TanStack Start
- TypeScript
- React
- Tailwind CSS
